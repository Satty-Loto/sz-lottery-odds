# sz-lottery-odds

**Калькулятор математических вероятностей лотерей АО «Сәтті Жұлдыз» (sz.kz)**

Открытый инструмент для точного расчёта вероятностей выигрыша, шансов и математического ожидания в лотереях казахстанского государственного оператора.

## Методология

Все вероятности рассчитываются точно через **биномиальные коэффициенты**:

```
P(m совпадений из k при n выбранных из N) = C(k,m) × C(N-k, n-m) / C(N,n)
```

Для Keno используется **гипергеометрическое распределение** (выборка без возвращения).

### Пример — Loto 6/49

| Категория | Вероятность | Шанс |
|-----------|-------------|------|
| 6 из 6 (Суперприз) | 0.0000000715% | 1 : 13 983 816 |
| 5 + бонус | 0.0000000429% | 1 : 2 330 636 |
| 5 из 6 | 0.0000018% | 1 : 55 492 |
| 4 из 6 | 0.000097% | 1 : 1 032 |
| 3 из 6 | 0.0177% | 1 : 57 |
| 2 из 6 | 0.1324% | 1 : 7.5 |

```
C(49,6) = 13 983 816 возможных комбинаций
Вероятность джекпота = 1 / 13 983 816 ≈ 7.15 × 10⁻⁸
```

### Пример — 5/36

```
C(36,5) = 376 992 возможных комбинаций
Вероятность джекпота = 1 / 376 992 ≈ 2.65 × 10⁻⁶
```

## Математическое ожидание

Ожидаемый возврат для большинства государственных лотерей составляет **0.40–0.55 тенге на каждый потраченный тенге**.

Это означает, что в среднем с каждых 1 000 ₸ ставок участник статистически получает обратно 400–550 ₸. Разница идёт на операционные расходы оператора и отчисления в бюджет.

> **Важно:** калькулятор создан для образовательных целей. Он показывает математическую реальность, не «стратегии» или «системы» — их не существует.
