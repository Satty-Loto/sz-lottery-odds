Методология расчётов

Все вероятности рассчитываются точно через биномиальные коэффициенты (комбинаторика). Приближения не используются.

// Биномиальный коэффициент C(n, k) — количество комбинаций C(n, k) = n! / (k! × (n-k)!) // Вероятность угадать ровно m чисел из k выбранных (лото k/N, выбрано n шаров) P(m совпадений) = C(k, m) × C(N-k, n-m) / C(N, n) // Пример: Loto 6/49, угадать 4 числа из 6 P = C(6,4) × C(43,2) / C(49,6) = 15 × 903 / 13 983 816 = 13 545 / 13 983 816 ≈ 1 : 1 032

Математическое ожидание

// E = сумма по всем категориям: вероятность × размер выигрыша // Для фиксированных призов (Keno, кат.6 в 6/49, кат.4 в 5/36): приз известен точно. // Для нефиксированных категорий: используем % от призового фонда. // Призовой фонд принят = 50% от суммы реализации тиража. // Интерпретация результата: // EV = 0.50 → на каждые 100 тенге ставки ожидаемый возврат = 50 тенге // EV = 1.00 → математический ноль (не бывает в государственных лотереях) // Типичный диапазон государственных лотерей: 0.40 – 0.55

Нефиксированные выигрыши в 6/49 и 5/36 зависят от числа победителей в данном тираже и не могут быть точно рассчитаны заранее. Калькулятор использует ориентировочные значения на основе нормального распределения участников.
